Forman Christian College (A Chartered University)
FACULTY PROMOTION PORTAL 

You are an expert AI system designer, higher education policy architect, and academic evaluation engine builder.

Design a complete Faculty Promotion Portal aligned with HEC Pakistan criteria for Associate Professor and Full Professor. The system must be rule-based, secure, dynamic, auditable, and capable of generating structured evaluation reports, eligibility decisions, and multi-layer reviewer feedback.

====================================================
A. OBJECTIVE
====================================================
Develop a comprehensive academic promotion system that:
- Allows secure account creation for faculty, internal reviewers, and external reviewers
- Enables draft creation, editing, and final submission workflow
- Separates Application for Promotion and Self-Evaluation Report
- Applies dynamic eligibility and scoring rules
- Supports document uploads and validation
- Evaluates teaching, research, scholarship, efforts to improve as teacher, and services
- Enables internal and external reviewer evaluation
- Generates structured reports and downloadable academic dossiers (PDF)

====================================================
B. APPLICATION STRUCTURE (TWO PART SYSTEM)
====================================================

1. APPLICATION FOR PROMOTION
- All sections are fully applicable
- Used for eligibility, scoring, and institutional decision

Mandatory Document Requirements (Section G only):
- Items 1, 2, 3, 5, 6, 8, and 9 must be uploaded before submission

2. SELF-EVALUATION REPORT
Only the following sections are included:
- Section G (No. 5, 6, 7, 9)
- Section H
- Section I
- Section J
- Section K

Mandatory Document Requirements (Section G only):
- Items 1, 2, 3, 5, and 9 must be completed

Applicant Note:
- Download this application and send it to your Chair for review

====================================================
C. USER ROLES (STRICT RBAC SECURITY)
====================================================

1. Faculty Member (Applicant)
2. Internal Reviewer
3. External Reviewer
4. Admin

Access rules:
- Faculty can only access their own application
- ALL internal reviewers can access ALL submitted applications
- External reviewers can ONLY access:
- Updated CV only
- Admin has full system access

Applicants MUST NOT access:
- Reviewer comments
- Scoring sheets
- External reviewer evaluations
- Internal analysis modules

====================================================
D. APPLICATION LIFECYCLE
====================================================

1. Registration/Login:
- Secure account creation for faculty
- Secure account creation for internal reviewer
- Secure account creation for external reviewer

2. Draft Mode:
- Save and update application anytime
- Upload documents progressively

3. Pre-Submission Review:
- Complete application preview
- Validation checklist (including Section G mandatory document checks)
- Final confirmation before submission

4. Final Submission:
- Application becomes read-only

5. Post Submission:
- Applicant can view and DOWNLOAD:
- Promotion Application (PDF)
- Self-Evaluation Report (PDF)

====================================================
E. PROMOTION CRITERIA
====================================================

ASSOCIATE PROFESSOR:
- Must pass Versant English Test (VET)
- PhD required
- Experience:
- 10 years teaching/research OR
- 5 years post-PhD experience
- Publications:
- Minimum 10 recognized journal articles

FULL PROFESSOR:
- Must pass Versant English Test (VET)
- PhD required
- Experience:
- 15 years teaching/research OR
- 10 years post-PhD experience
- Publications:
- Minimum 15 recognized journal articles

Publication

o Journal name

o Issue
o Article title
o Number of authors (dropdown) 
o Author names (auto fields)

o "Add another publication" button
o Publication count

Recognized indexing sources:
- HEC recognized journals
- Scopus indexed journals
- Web of Science (WoS)
- DOAJ (Directory of Open Access Journals)
- CNKI

System must validate journal indexing and flag unrecognized publications.


APA Generator

====================================================

- Not Recommended

- Recommended
